Movies App Demo

This repository contains the source code for http://movies-app-demo.s3-website-us-west-2.amazonaws.com.

It uses React with webpack to build a Single Page Application.

Resolutions

Resolutions above 1024 should work. Also optimized for HD and Full HD TV resolutions.

Setting up

Prerequisites

  1. Git
  2. Node: versions v6.14.4, v8.12.0, v9.11.2, v10.12.0, 11.10.1, v12.4.0 are tested and running as expected. Any versions for v9.x, v10.x, v11.x and v12.x should work

Instalation

  1. git clone https://github.com/iepsen/movies-app-demo.git
  2. cd movies-app-demo
  3. npm install

Running

  1. npm start

npm start uses webpack-dev-server with default configuration and will load on port 8080.

Tests

  1. npm test

Only presentation layer components are covered. More tests could be added soon.

Hosting

This project flows the Github Actions pipeline to test, build and deploy it on Amazon AWS if all tests pass.

See the configuration file for more details.
